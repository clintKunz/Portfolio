Projects

Bevy Loans

Business case: 57.7% of surveyed adults have borrowed more than $500 from a family member or friend. 60.5% of these borrowers don't keep an official record of terms, expectations, and repayment. Bevy's purpose is to facilitate those loans in a simple way that provides transparency and accountability to both parties.

Technical highlight: Implementing Redux, a state management tool, into this project has several technical benefits. 1) Data is predictable and ease to access from any component and 2) Debugging and maintenance are easier because of Redux's strict structure.

Tech stack: React.js, Redux, MongoDB, Node.js

Lambda Notes

Business case: A note taking app that allows users to create, read, update, and delete notes.

Technical highlight: This project was built with a SQL database because it was important to protect the integrity of the database and the data was designed to be structured and unchanging.

Technical stack: React.js, PostgreSQL, Node.js
